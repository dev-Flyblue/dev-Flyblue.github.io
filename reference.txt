### Jekyll + Chirpy 最新安装 & 起步指南（2026年3月最新）
官方文档主要在：https://chirpy.cotes.page/ 和 GitHub Wiki https://github.com/cotes2020/jekyll-theme-chirpy/wiki

#### 1. 前置要求
- 安装 **Ruby**（≥ 2.7.0，推荐 3.x 版本）
- 安装 **RubyGems**、**Jekyll** 和 **Bundler**
- 安装 **Git**

按操作系统：
- **macOS**：用 Homebrew `brew install ruby`，然后 `gem install jekyll bundler`
- **Windows**：安装 RubyInstaller（带 DevKit），然后同上 gem install
- **Linux**（Ubuntu/Debian）：`sudo apt update && sudo apt install ruby-full build-essential zlib1g-dev`，然后 gem install

官方 Jekyll 安装：https://jekyllrb.com/docs/installation/

安装完验证：
```bash
ruby -v
gem -v
jekyll -v
```

#### 2. 最推荐方式：用 Chirpy Starter（官方推荐，升级简单）
这是零基础最友好方式，不用手动 gem install 主题。

步骤：
1. 登录 GitHub → 访问 https://github.com/cotes2020/chirpy-starter
2. 点击绿色 **Use this template** → **Create a new repository**
3. 仓库名必须是：**你的用户名.github.io**（比如 shhn.github.io），这样才能自动用 GitHub Pages
4. 创建后，**Clone** 到本地：
   ```bash
   git clone https://github.com/你的用户名/你的用户名.github.io.git
   cd 你的用户名.github.io
   ```

5. 安装依赖：
   ```bash
   bundle install
   ```

6. 本地运行预览（超级重要，先看效果）：
   ```bash
   bundle exec jekyll serve
   ```
   浏览器打开 http://127.0.0.1:4000 就能看到博客了！

7. 写第一篇文章：
   - 在 `_posts` 文件夹新建 Markdown 文件，命名格式：`YYYY-MM-DD-文章标题.md`
   - 文件开头加 Front Matter（必须）：
     ```yaml
     ---
     title: "我的第一篇博客"
     date: 2026-03-10 03:00:00 +0800
     categories: [生活]
     tags: [hello-world]
     ---
     正文用 Markdown 写，支持 LaTeX 公式、Mermaid 图表等。
     ```

8. 提交 & 推送：
   ```bash
   git add .
   git commit -m "Add first post"
   git push origin main
   ```

9. 开启 GitHub Pages：
   - 仓库 Settings → Pages → Source 选 main 分支 → Save
   - 等 1-2 分钟，访问 https://你的用户名.github.io 就上线了！

#### 3. 自定义小Tips（Chirpy 很强大）
- 编辑 `_config.yml` 改标题、描述、社交链接、avatar 等
- 支持暗黑模式（默认就有）
- 加数学公式：用 `$$...$$` 或 `$...$`
- 加流程图：用 Mermaid 语法
- 自定义域名：买个 .top/.xyz 域名（阿里云/Namesilo 几十块），然后在仓库 Settings → Pages 加 Custom domain
- 国内加速：用 Cloudflare Pages 部署（fork Chirpy 项目，导入到 Cloudflare，速度更快）
